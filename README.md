# Reunite

**Reunite** es una aplicación web diseñada para simplificar la organización de reuniones entre amigos, familiares o compañeros. 
Permite coordinar fechas, lugares, tareas y gastos de forma colaborativa, integrando servicios externos para una experiencia completa y sin complicaciones.

---

## Características principales

- **Creación y gestión de reuniones:** Define título, descripción, fecha, hora y ubicación.
- **Invitaciones:** Envía invitaciones a tus amigos y gestiona confirmaciones (aceptar/rechazar).
- **Lista de ítems:** Crea elementos necesarios para la reunión y asigna quién lleva cada uno.
- **División de gastos:** Registra gastos y reparte el total equitativamente entre los participantes.
- **Votaciones:** Realiza encuestas para decidir fechas, lugares o cualquier otro aspecto de la reunión.
- **Notificaciones en tiempo real:** Mantén a todos informados ante cualquier cambio.
- **Integración con Google Maps:** Selecciona y visualiza la ubicación de la reunión en un mapa interactivo.
- **Integración con Gmail:** Envía invitaciones y notificaciones por correo electrónico.
- **Sugerencias climáticas:** Consulta el pronóstico del tiempo para recomendar cambios de plan si es necesario.

## Tecnologías utilizadas

| Tecnología | Propósito |
|------------|-----------|
| **Java 17** | Lenguaje de programación backend. |
| **Spring Boot 4.x** | Framework para API REST y gestión de dependencias. |
| **Apache Tomcat 10** | Servidor de aplicaciones. |
| **MySQL** | Base de datos relacional. |
| **Git & GitHub** | Control de versiones. |
| **Jira** | Gestión de proyectos (metodología ágil). |
| **Postman** | Pruebas de API. |
| **Draw.io** | Diagramas UML. |

### APIs externas
- **Google Maps API** – Para mapas, geolocalización y sugerencias de puntos de encuentro.
- **Gmail API** – Para envío de correos de invitación y notificaciones.
- **OpenWeatherMap API** (u otra) – Para pronóstico del clima y recomendaciones.
